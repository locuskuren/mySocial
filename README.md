# mySocial social network platform

![image](https://user-images.githubusercontent.com/94398533/174828275-0400dd31-4a0b-4673-b296-a2548d2ea891.png)

## Built with:

- Typescript
- React
- Redux
- Redux Thunk
- Axios
- React Router
- Sass

## Brief Description:

Simple social network platform on which you can:
 - authenticate
 - create posts
 - like posts
 - delete posts
 - follow other users
